# 🤰 Multimodal Pregnancy Risk Prediction System

A multimodal AI system that predicts pregnancy risk by combining **ultrasound images** and **maternal clinical data**. The project uses deep learning and machine learning models to classify pregnancies as **High Risk** or **Low Risk**.

## 🌐 Live Demo

🔗 https://pregnancy-predictor-835v.vercel.app

---

## ✨ Features

- Predicts High Risk and Low Risk pregnancies
- Combines ultrasound images with maternal clinical data
- CNN-based image feature extraction
- Machine learning-based clinical data analysis
- Multimodal feature fusion for improved prediction
- User-friendly web application for real-time prediction

## 🚀 How It Works

1. Upload a fetal ultrasound image.
2. Enter maternal clinical information.
3. The image and clinical data are processed separately.
4. Their features are combined using a fusion model.
5. The system predicts whether the pregnancy is **High Risk** or **Low Risk**.
